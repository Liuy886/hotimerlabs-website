# HotimerLabs Web3 官网话术规范

## 原则

官网文案必须贴近 Web3 项目方、交易所、做市商、KOL/PR 资源方真实沟通语境，避免为了“看起来高级”而创造不常见词汇。

## 避免使用的表达

这些词不是不能用，但在官网首屏不建议作为核心术语：
- Mandate Console
- Confidential Execution Layer
- Capability Map
- Market-side view
- Strategic Advisory Lab
- Verified growth paths
- Market path

原因：它们更像包装词，不是行业客户搜索、沟通、采购服务时会自然使用的词。
