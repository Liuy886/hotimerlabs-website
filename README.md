# HotimerLabs Website

HotimerLabs bilingual Web3 consulting landing page.

## Preview

Run a local static server from this folder:

```bash
python3 -m http.server 4173
```

Open:

```text
http://127.0.0.1:4173
```

## Files

- `index.html`: page structure and SEO metadata
- `styles.css`: responsive layout and Institutional / Native visual modes
- `script.js`: language switching, visual mode switching, Telegram lead flow
- `assets/hotimerlabs-hero.png`: generated hero visual asset
- `website-product-plan.md`: original product plan
- `website-product-plan.zh.md`: Chinese product plan

## Telegram

The current Telegram handle is configured in `script.js`:

```js
const telegramHandle = "HotimerLabs";
```

Replace it with the final public Telegram username before launch.
